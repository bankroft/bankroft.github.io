centos7.2下php7.0+mysql5.7+apache环境

安装apache

yum install httpd

安装php7.0

参考文章

添加源

rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm
rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm

安装php7.0

yum install php70w php70w-opcache php70w-mysql -y
yum install php70w-pdo php70w-xml php70w-mbstring -y
yum install php70w-mcrypt php70w-gd -y

安装MySQL

MySQL5.7下载地址

mysql

wget -O mysql5.7.tar https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.20-1.el7.x86_64.rpm-bundle.tar
