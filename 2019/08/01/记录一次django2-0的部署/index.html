[安装python3.6]

安装virtualenv
pip3 install virtualenv
启用env等

安装dj、mod_wsgi等
pip install mod_wsgi django==2.0
报错:
MySQL config
安装MySQL devel可解决 yum install mysql-devel

安装apache2.4
yum install httpd

./configure --with-python=/usr/local/python3.6
make && make install
