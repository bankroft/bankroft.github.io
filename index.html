超星刷课

题库

开放query api和update api，手里有题库的欢迎post

查询api - 开放(查询频率5/m)

api

http://123.207.19.72/api/query

方法

GET

参数
  • title:题目(不包含题目类型)
  • enc:验证位
返回json

{code:'', data: ''}

查询api - token(查询频率依token而定,个人用户可从群文件获取token)

api

http://123.207.19.72/api/token-query

方法

GET

参数
  • title:题目(不包含题目类型)
  • token:token(联系博主获取)
返回json

{code:'', data: ''}

update api

api

http://123.207.19.72/api/update

方法

POST

参数
  • title: 题目(不包含题目类型)
  • answer:题目答案(单选题为选项内容,多选题选项内容之间用&分割,判断题用True or False或者√ or ×)
  • enc:验证位
    返回json

{code: ''}

enc

md5(title + answer(如果有) + '-b?M#JvMg2y3$JMk')

code

  • 100: 操作正常(query有结果，update成功)
  • 101：操作正常(query未查到)
  • 404：请求错误

使用方法：

使用

  1. 安装Chrome，版本为last stable
  2. 下载最新版软件压缩包并解压
  3. 双击行解压出的exe文件

第一次运行

  1. 运行软件后选择 功能3：设置
  2. 配置答题微信公众号(暂不可用)，可以直接回车跳过；如果填了，先关注该公众号，软件启动的时候弹出的二维码使用微信扫描
  3. 配置查题token(可选)
  4. 配置默认播放线路(因存在部分学校默认线路无法播放)，可以直接回车跳过
  5. 配置若快打码平台账号和密码，可以直接回车跳过

注：

设置选项只是修改软件包内config.ini文件，如果软件有更新，请用旧版本内config.ini替换新版本，或者重新运行设置选项
